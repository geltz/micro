micro is a granular sampler and effects processor.	

**controls**

left mouse for region selection.    
middle/right mouse to pan on waveform.  

**buttons**
- clear
- export  

**toggles** 
- reverse

**sliders**
- attack
- release
- pan
- reverb
- click
- rate
- tone    
- crush
- shake
- grain

this program is free and [donations are welcome](https://ko-fi.com/geltz)

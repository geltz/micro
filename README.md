micro is a waveform-centered sound engine and effects processor with a qt6 interface.	

it isolates regions of an audio sample and applies various effects.	

**controls**

left mouse for region selection.    
middle/right mouse to pan on waveform.  

**buttons**
- clear
- export  

**toggles**
- click (impulses)
- reverse
- grain  

**sliders**
- attack / release
- pan
- reverb
- playback rate
- tone (filter)
- crush
- compress

micro is a waveform-centered sound engine and effects processor with a pyqt6 interface.	

it isolates regions of an audio sample and applies various effects.	

**buttons**
- clear
- click (short impulses)
- reverse
- repeat (loop playback)
- export wav

**sliders**
- attack / release
- pan
- reverb
- playback rate
- tone (filter)
- crush
- compress
